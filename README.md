# GRC-Automation-Toolkit
A practical, code‑light but high‑impact project that demonstrates governance, risk, and compliance maturity.
The GRC Automation Toolkit is a collection of small, modular tools designed to streamline and automate everyday Governance, Risk, and Compliance tasks.
It reflects real‑world GRC work: risk scoring, policy lifecycle management, control mapping, and vendor assessments all implemented in simple, transparent scripts.

This project demonstrates how security governance can be practical, data‑driven, and automation friendly, making it ideal for security teams, GRC analysts, and organizations looking to mature their security processes.

Automated Risk Scoring Engine  
Generates consistent, repeatable risk scores for assets, vendors, or findings.

Policy Version Tracker  
Tracks policy updates, version history, and approval workflows.

Control Mapping Utility  
Maps controls across frameworks (ISO 27001, NIST CSF, CIS, SOC 2).

Vendor Risk Assessment Helper  
Processes questionnaire responses and produces a risk summary.

Simple, Extensible Architecture  
Each module is standalone and easy to customize.
